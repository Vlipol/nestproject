# nestproject
Сервіс з генерування рецептів

Допомагає користувачам знаходити рецепти страв за наявними інгредієнтами
Користувач вводить продукти, які має вдома, а система підбирає можливі страви, використовуючи відкритий API

Мета проєкту:
Створити зручний сервіс, який:
1) економить час користувача
2) допомагає ефективно використовувати продукти
3) навчає новим кулінарним ідеям

Функціональні можливості:
1) Пошук рецептів за списком інгредієнтів
2) Перегляд детальної інформації про рецепт (назва, опис, кроки приготування, калорійність)
3) Використання зовнішнього API
